## GameJAM Player

GameJAM Player is a lightweight sandbox I use to iterate on jam-sized game ideas. It focuses on:

- zero-ceremony scene bootstrapping so I can focus on the event loop, not boilerplate
- rapid asset swapping with hot-reloadable sprites and palettes
- deterministic scoring loops that make balancing short sessions painless

You can try a tiny HTML playground that mirrors the feel of my jam prototypes right from the browser:

- [Play the GameJAM Player mini-demo](https://mairuis.github.io/gamejam-player.html)
